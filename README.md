# Divine Ressha Frontend

This Next.js app recreates the Divine Ressha landing page from the live site at `https://world-cart-bloom.lovable.app`.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```

## Project structure

- `app/` - Next.js App Router files
- `app/page.tsx` - Landing page content and layout
- `app/layout.tsx` - Root layout and metadata
- `app/globals.css` - Global styling for the site
